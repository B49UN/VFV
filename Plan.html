Let's build a website for 'Synsemanti Reader'

We need these fuctions:

(Two pages)

1. File upload (pdf, jpg, png)
2. Save the file to the Database using a web server
3. Bring the analyzed version data of the according uploaded one from the server
4. Display

Obstacles:

  1. 2단계는 현재 서버가 없어서 파일 업로드하고 등록 누르면 에러 뜸, 서버 구축 필요

  2. 3단계에서 output data를 database에서 server를 통해 어떻게 읽어올 것인가(SQL, PHP), 현재는 database data가 아닌 이미지 링크이기 때문에 다운로드 버튼 눌러도 리디렉션될 뿐

  3. 4단계에서 output data는 어떤 형태로 display 될 것인가(text data를 어떤 형태로 전환하여야 하는가)

  4. Database에 저장된 input data를 'pdf reading하여 text로 변환 & API로 분석'한 뒤 산출된 output data 다시 database에 저장하는 내부 과정 필요(nondisplayed)
